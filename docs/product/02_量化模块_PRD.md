## 0. 一页纸

### 做什么
InvestDojo 量化模块 —— 一个能让**散户 / 量化爱好者 / 策略研究员**在同一平台上：浏览因子 → 训练模型 → 回测验证 → 与自己 PK 的完整工具链。

### 为什么
市面上量化平台（聚宽、米筐、Qlib）和散户模拟盘（同花顺、雪球）是**两个割裂的世界**：
- 量化圈：只看回测数字，不体会市场"怕"的感觉
- 散户圈：只有情绪和 K 线，不懂因子和模型

InvestDojo 把"模型做出的量化判断"和"人在真实情境里做出的决策"放在**同一屏同一时刻**让两者打架——这是"道场"二字最好的注解。

### 北极星指标
**人均月盲测次数 × 人均每次盲测加载模型数**
= 衡量"用户真的在把模型放进真实情境对比"的核心指标

## 1. 用户画像与场景

### 1.1 四类画像

| 画像 | 能力 | 痛点 | 在 InvestDojo 解决 |
|------|------|------|-------------------|
| **散户小白** | 只懂 K 线 | 不知道别人怎么赚钱 | 因子库一键选 → 一键训模型 → 副驾陪练 |
| **量化新手** | 会 Python | 没地方"真实"验证策略 | 上传模型 → 进盲测环境 → 对比人的决策 |
| **策略研究员** | 专业 | 想观察人在真实情境的反应 | Notebook 自由造因子 → PK 模式 → 数据闭环 |
| **沉默大多数** | 只想看结果 | 不想学 Python | 模型市场套用别人的策略 |

### 1.2 典型场景

**场景 A · 散户小白小李**
> 小李玩了几次盲测，一直输给市场。他点开量化模块，发现"推荐因子"里有"20日均线上穿60日均线"。他选了这个加几个技术指标，点训练，3 分钟后得到一个胜率 56% 的模型。他开启副驾模式再做一次盲测，每次下单前都看看模型怎么说。他发现自己经常和模型反着做——意识到自己在跟感觉走。

**场景 B · 量化新手小红**
> 小红在自己的电脑上研究出了一个动量策略，用 LightGBM 训了一个模型。她把 `.pkl` 文件上传到 InvestDojo，开启 PK 模式，选了一段随机的历史时期。60 天后模型赢了她 +2.3%，但最震撼的是"决策一致度仅 47%"——她看到了自己和自己的模型之间的巨大分歧，开始反思自己的交易偏好。

**场景 C · 研究员老王**
> 老王在 Notebook 里用全量 A 股 2014-2025 数据研究了一个"财报公告后 3 日异动"因子。他把因子发布到平台，有人复用，他看到了自己的因子在 50 个别人的盲测里的表现分布。他发现这个因子在牛市阶段失效——这是他在自己的研究里没观察到的。

**场景 D · 沉默大多数老张**
> 老张从不写代码。他在"模型市场"按"近 3 个月收益"排序，看到一个用户发布的 "量价背离模型 v3" 有 +15% 收益，他点"在盲测中试用"，系统把这个模型作为副驾陪他过了 60 天。结束后他理解了这个模型的风格，决定持续使用。

## 2. 功能清单与优先级

### 2.1 功能全景图

```
【特征工程】
├─ P0  因子库浏览（200+ 内置因子）
├─ P0  因子详情页（公式/历史表现/示例）
├─ P1  可视化因子组合器（拖拽 + 公式）
└─ P2  Notebook 集成（浏览器内 Python）

【模型训练】
├─ P0  平台内一键训练（LightGBM）
├─ P1  更多算法（XGBoost/LSTM/Transformer）
├─ P1  用户上传模型（pkl/onnx/pt）
└─ P2  第三方信号导入（聚宽/米筐/Qlib）

【模型市场】
├─ P1  公开模型排行榜
├─ P1  平台官方模型（⭐ 新需求）
├─ P2  模型订阅/付费
└─ P2  模型使用量统计

【回测】
├─ P0  向量化快速回测
├─ P0  评估报告（雷达图/净值/分段/特征重要性）
├─ P1  事件驱动精细回测（滑点/手续费/撮合）
└─ P2  多策略对比

【联动模式】
├─ P0  模式① 副驾（AI 建议但不下单）
├─ P0  模式④ 数据闭环（样本收集，先不训）
├─ P1  模式③ Copilot（采纳/拒绝日志）
└─ P1  模式② PK（人机对抗）

【监控】
├─ P1  模型表现追踪
├─ P1  数据漂移告警
└─ P2  调用量/延迟监控

说明：P0 = MVP 必须 / P1 = 第二版 / P2 = 未来迭代
```

### 2.2 MVP 范围（P0 全集）

```
✅ 因子库浏览 + 详情页
✅ 平台内一键训练（LightGBM）
✅ 向量化回测 + 评估报告
✅ 模式① 副驾
✅ 模式④ 数据收集
```

MVP 不做：模型上传、PK、Copilot、Notebook、可视化组合器、第三方信号、付费机制、排行榜。

## 3. 核心用户故事

### 3.1 因子库（P0）

**US-F01** · 浏览因子
> 作为小李，我想按类别浏览平台提供的因子，看每个因子的名字、用途、简单表现，这样我能快速找到感兴趣的因子放进模型。

验收标准：
- 因子按 技术 / 估值 / 成长 / 情绪 四大类展示
- 每个因子卡片显示：名称 / 一句话描述 / 历史触发次数 / 触发后 20日平均收益
- 支持按分类筛选、按热度/胜率排序
- 可加入"我的收藏"

**US-F02** · 查看因子详情
> 作为小红，我想看因子的完整定义（公式）、在历史上的分布、典型触发案例，这样我能判断是否适合自己的策略。

验收标准：
- 展示因子的数学公式（LaTeX 渲染）和 Python 代码
- 过去 5 年的触发次数时序图
- 触发后 5/10/20 日的收益率分布直方图
- 最近 10 次触发案例（带 K 线缩略图）
- "在回测中试用" 快捷按钮

**US-F03** · 创建自定义因子（P1，可视化组合器）
> 作为小红，我想拖拽几个已有的指标和运算符造一个复合因子，不想写 Python。

验收标准：
- 提供拖拽画布 + 公式编辑器
- 实时预览因子在某只股票上的历史表现
- 保存到"我的因子"并可复用
- 可"一键转为 Python 代码"看看等价实现

---

### 3.2 模型训练（P0）

**US-M01** · 一键训练
> 作为小李，我想在 5 分钟内训练出一个可用的模型，不需要懂算法。

验收标准：
- 四步向导：选因子 → 选预测目标 → 选时间范围 → 选算法
- 每步都有推荐默认值（新手直接下一步也能成）
- 训练时展示进度和预计剩余时间
- 训练完成邮件/站内信通知
- 最长训练时间不超过 10 分钟

**US-M02** · 查看训练报告
> 作为小红，我想看模型的各种评估指标，判断模型的强弱和适用场景。

验收标准：
- 核心指标卡片：准确率 / AUC / IC / 夏普 / 回撤
- 分段表现：牛市/熊市/震荡市下分别怎样
- 特征重要性 SHAP 图
- 验证集上的混淆矩阵
- 提供"与其他模型对比"入口

**US-M03** · 上传模型（P1）
> 作为量化新手，我想把自己训好的模型上传到 InvestDojo，不想重新训练。

验收标准：
- 支持 pkl/onnx/pt/h5 四种格式
- 上传时声明特征列表
- 上传后在沙箱环境跑一次 smoke test
- smoke test 通过后自动做一次简单回测，展示报告
- 模型体积限制 500MB

---

### 3.3 模型市场（P1）

**US-X01** · 浏览公开模型
> 作为老张，我想看平台上其他用户发布的模型，按收益/胜率排序挑一个用。

验收标准：
- 列表显示：模型名 / 发布者 / 近 3 月表现 / 调用次数 / 星级
- 支持按类别（趋势/反转/量价/基本面）筛选
- 可按多维度排序：收益 / 稳定性 / 回撤 / 热度
- 点进去能看详情和一键试用

**US-X02** · 平台官方模型（⭐ 新需求）
> 作为老张，我想看 InvestDojo 官方维护的"基线模型"——它们不是最强的，但代表不同风格的典型策略。我能看到它们的完整方法论。

验收标准：
- 模型市场顶部专门设有"⭐ 官方模型"板块
- 至少 5 个官方模型，覆盖不同流派：
  - "动量追踪者"（技术指标为主）
  - "价值投资者"（估值+成长因子）
  - "情绪交易者"（换手率+北向资金）
  - "多因子合成"（稳健型基线）
  - "深度学习实验者"（LSTM/Transformer）
- 每个官方模型：
  - 有详细文档（设计思路、历史表现、局限性）
  - 开源特征列表和训练代码（可复用）
  - 持续更新（每季度重训一次）
  - 不能删除、不能私有
- 官方模型可作为新用户的"默认副驾"

**US-X03** · 订阅模型（P2）
> 作为老张，我想把某个模型设为默认副驾，以后每次盲测都自动加载。

**US-X04** · 模型付费（P2）
> 作为老王，我想对自己发布的高质量模型设定付费（如每次调用 ¥0.5），别人用我能有收益。

---

### 3.4 回测（P0）

**US-B01** · 一键回测
> 我想在模型训练完成后立即跑一次完整回测，看它在训练集外的表现。

验收标准：
- 训练完成自动触发回测
- 回测时间范围 ≠ 训练时间范围
- 回测结果在 30 秒内返回
- 默认参数：初始资金 10 万、手续费万三、滑点千一

**US-B02** · 对比基准
> 我想看我的模型相对上证指数/沪深300/等权基准的超额收益。

验收标准：
- 净值曲线默认叠加上证和沪深300
- 关键指标给出"相对值"（如 vs 上证 +12.3%）

---

### 3.5 联动模式（P0-P1）

#### 模式① 副驾（P0）

**US-L01** · 开启副驾
> 我想在盲测模式中开一个 AI 副驾面板，每次切 K 线它都告诉我它的判断。

验收标准：
- 盲测配置页新增"副驾模型"下拉选择
- 默认推荐一个平台官方模型
- 可切换模型中途替换
- 副驾面板显示：当前信号 / 置信度 / 关键理由（TOP 3 特征）
- 副驾不能替用户下单

#### 模式② PK（P1）

**US-L02** · 与模型对战
> 我想和自己训的模型玩一局 PK，看谁表现更好。

验收标准：
- 人和模型拿相同本金、看同一份数据
- 模型每 tick 必须在 5 秒内给决策，超时视为持有
- 结束后生成对账报告：收益/回撤/夏普/交易次数
- 特别展示"决策一致度"和关键分歧点
- 支持"看每一笔分歧"的深度复盘

#### 模式③ Copilot（P1）

**US-L03** · 协作下单
> 我想每次下单前让模型先建议，我决定采纳还是拒绝。

验收标准：
- 下单确认弹窗增加"模型建议" 区域
- 三个选项：完全采纳 / 微调 / 完全拒绝
- 记录每次选择到"决策日志"
- 积累 40+ 次会话后生成个性化洞察报告

#### 模式④ 数据闭环（P0 仅收集）

**US-L04** · 会话数据入库
> 我的每次盲测操作自动保存为训练样本，未来可用于训练"我的数字孪生"。

验收标准：
- 所有盲测会话默认收集样本（可在设置里关闭）
- 每个样本包含：特征/标签/上下文
- 样本默认私有，只自己能看
- 可查看自己累计的样本数量

## 4. 非功能需求

### 4.1 性能
- 因子库浏览：首屏 < 1s
- 因子详情页：< 2s
- 一键训练：< 10 分钟
- 向量化回测：< 30s
- 副驾信号延迟：< 200ms

### 4.2 容量
- 单用户可保存模型：20 个
- 单用户上传模型文件：500MB
- 单用户盲测样本：无限
- 平台官方模型：长期保留

### 4.3 隐私
- 用户上传的模型代码绝对私有，平台不可读
- 用户的盲测样本默认私有
- 贡献到公共池必须显式同意 + 严格脱敏
- 可随时删除自己的数据

### 4.4 合规
- 不提供投资建议（首页/模型详情都有免责声明）
- 模型名称不得使用"稳赚/保本/无风险"等词
- 付费模型需标注"历史表现不代表未来"

## 5. 成功指标

### 5.1 激活指标
- 新用户训练第一个模型的转化率 > 30%
- 新用户 7 日内做 1 次副驾盲测的比例 > 50%

### 5.2 留存指标
- D30 留存 > 25%
- 月活用户人均模型数 ≥ 3

### 5.3 核心指标（北极星）
- **人均月盲测次数 × 人均每次盲测加载模型数** 持续增长
- 此指标反映"人机联动"这件事的真实发生频率

### 5.4 质量指标
- 平台官方模型年化收益 vs 上证差值 > +5%（代表平台的基准能力）
- 用户发布的公开模型中，月调用量 > 100 的比例 > 10%
