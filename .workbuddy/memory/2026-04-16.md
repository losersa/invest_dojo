# 2026-04-16 工作日志

## InvestDojo 招商 PPT 多风格变体生成
- 基于原版 `investdojo-pitch.html`（Apple Design System 风格），生成了 3 种全新风格的可编辑版本：
  1. **Bold Signal** (`investdojo-pitch-bold.html`) — 大胆橙色强调、Archivo Black 字体、高冲击力
  2. **Swiss Modern** (`investdojo-pitch-swiss.html`) — 包豪斯极简、直角边框、网格背景、红色点缀
  3. **Dark Botanical** (`investdojo-pitch-botanical.html`) — 优雅暗色、Cormorant 衬线字体、暖色柔光
- 所有版本均支持：
  - 浏览器内文字编辑（左上角 hover 出 ✏️ 按钮，或按 E 键）
  - Ctrl+S 导出干净 HTML（自动剥离编辑状态）
  - 键盘←→翻页、触摸滑动、鼠标滚轮
  - 右侧竖向圆点导航 + 顶部进度条
  - 全部使用 clamp() 响应式排版（viewport-base 规范）
- 使用 frontend-slides Skill 的模板规范和动画模式
